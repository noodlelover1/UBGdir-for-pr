Rules:

  1. The site must have at least 20 games.
  2. The site must not contain any malicious ads or links such as viruses, scams etc.
  3. The site should have some feature that reduces the risk of it being blocked.
  4. The site must be suitable for all ages
  5. You must own or have owners permission of the site you want to submit.
  6. optional: The site should not use external embeds.

Tips for beginner devs

  • You can find single file games on the ultimate game stash.
  • Keep your site optimized for slower devices and avoid heavy javascript use.
  • Keep backups and be prepared, some freehosts like neocities might remove your site and you might face other problems later on.
  • Try to keep it unblocked, add tools and extra features like about:blank and cloaking and not just dump a bunch of games.
  • update it often and use a contact option to get user feedback.
